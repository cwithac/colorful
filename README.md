# Color Thesaurus

### [Hosted on Heroku](https://colorthesaurus.herokuapp.com/)

![](http://i.imgur.com/kW6Bx5U.png)

### Technologies Used

```
+ Node.js, Express, EJS
+ Mongo DB, Mongoose
+ MVC Structure | Models - Colors, Words
+ Partials: head, nav
+ RESTful routes: Complete - Words | Partial - Colors
+ CSS | Responsive design
```
